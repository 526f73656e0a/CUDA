This is a CUDA repository from my GPU_Programmierung course. 
The code has benn tested and is working for Ubuntu 18.04 and higher.
How to run-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Download the skeleton code and generate the build system using cmake
  - Under Linux and MacOS you can use a package manager (apt, brew, port,. . . )to install cmake. Then type cmake .. in a directory ./build on thecommand line to generate the make file.  
  - On Windows cmake -G ‘‘VisualStudio 12’’ .\buildgenerates a Visual Studio solution.
For Assignment 3 an NVIDIA GPU Computer is required as well as the latest CUDA Toolkit
